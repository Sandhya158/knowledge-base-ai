# AI Knowledge Base Agent

A document-aware assistant that can read uploaded PDFs and answer questions with context, provide summaries, and maintain conversation history.

## Features
- Multi-PDF understanding
- Chat-style Q&A
- Instant summaries
- Confidence scoring
- Customizable assistant tone
- FAISS vector search

## Run
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
streamlit run app.py
